here is brief summary of this project

Technologies Used: Node.js,express.js,mongodb

Features:

  1. For authentication, I have used JSON Web Token (jsonwebtoken).
  2. It has an email feature: On successful orders, the user will be sent an email including the order details.
  3. it has wishlist feature, user can add products to wishlist
  4. it has Cart feature, user can add products to Cart

Postman : click here to open postman documentation

User and Auth

  1. to register the api endpoint is "/api/user/register"

  2. to login the api endpoint is "/api/user/login"

  3. to update the api endpoint is "/api/user/update"

  4. to own profile the api endpoint is "/api/user/my-profile"

  5. to add product to wishlist the api endpoint is "/api/user//add-to-wishlist/:id"

  6. to fetch wishlist the api endpoint is "/api/user/get-wishlist-products"

  7. to delete address the api endpoint is "/api/user/delete-address/:id"

Products

  1. to add products the api endpoint is "/api/products/post-product"

  2. to get all products the api endpoint is "/api/products/get-all-products"

  3. to get-a-product the api endpoint is "/api/products/get-a-product/:id"

Brands

  1. to get all brands the api endpoint is "/api/brands/get-all-brands"

Categories

  1. to get all Categories the api endpoint is "/api/category/get-all-categories"

Cart

  1. to get all Categories the api endpoint is "/api/cart/add-to-cart"

  2. to get all Categories the api endpoint is "/api/cart/get-all-cart"

  3. to get all Categories the api endpoint is "/api/cart/delete-cart-item/:id"

  4. to get all Categories the api endpoint is "/api/cart/update-cart-item/:id"

Orders

  1. to get all orders the api endpoint is "/api/orders/get-my-orders"

  2. to get single order the api endpoint is "/api/orders/get-my-order/:id"
